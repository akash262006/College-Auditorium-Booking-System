\# College Auditorium Booking System



A Java Swing based desktop application developed to manage college auditorium events and seat bookings.



\## Features

\- Faculty Login

\- Add Event and Slot

\- Remove Event

\- View All Bookings

\- Student Login and Registration

\- View Events

\- Seat Selection with Blocking

\- View My Bookings



\## Technologies Used

\- Java

\- Java Swing

\- JDBC

\- MySQL

\- Eclipse IDE



\## Database

\- Database Name: auditorium\_booking

\- Tables: admin, users, movies, showtimes, bookings



\## Database Configuration Note



For security reasons, the MySQL password is not included in the repository.



Before running the project:

1\. Open `DBConnection.java`

2\. Replace `YOUR\_MYSQL\_PASSWORD` with your local MySQL password

3\. Ensure MySQL server is running





\## How to Run the Project

1\. Import the project into Eclipse

2\. Configure MySQL database

3\. Update DBConnection.java with DB credentials

4\. Run LoginGUI.java
